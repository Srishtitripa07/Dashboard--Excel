# Dashboard--Excel
This project is an interactive Sales Performance Dashboard built in Microsoft Excel to analyze and visualize sales data across multiple cities and sales executives.
# Project Overview

The dashboard provides a complete overview of:

1. Total Sales by Sales Executive

2. City-wise Sales Filtering (Chennai, Delhi, Mumbai, Nagpur, Patna, Pune, Ranchi, Surat)

3. Target Hit Percentage (%)

4. Gap Analysis (Away From Target %)

5. Visual comparison using Bar Chart, Pie Chart, and Line Chart

It enables quick performance tracking and helps identify top performers and areas needing improvement.
# Key Features

a. City Slicer Filter – Dynamically filter data by city.

b. Bar Chart – Displays total sales comparison among sales executives.

c. Pie Chart – Shows target hit percentage distribution.

d. Line Chart – Visualizes how far each executive is from achieving the target.

e. Structured summary tables for better clarity.

f. Performance tracking based on predefined sales targets.
